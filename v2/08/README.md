# Задание 2 - форма обратной связи

В HTML есть разметка формы. Напиши скрипт который будет сохранять значения полей
в локальное хранилище когда пользователь что-то печатает.

```html
<form class="feedback-form" autocomplete="off">
  <label>
    Email
    <input type="email" name="email" autofocus />
  </label>
  <label>
    Message
    <textarea name="message" rows="8"></textarea>
  </label>
  <button type="submit">Submit</button>
</form>
```

Выполняй это задание в файлах `02-feedback.html` и `01-feedback.js`. Разбей его
на несколько подзадач:

1. Отслеживай на форме событие `input`, и каждый раз записывай в локальное
   хранилище объект с полями `email` и `password`, в которых сохраняй текущие
   значения полей формы. Пусть ключ для хранилища называется `form-state`.
2. При загрузке страницы проверяй состояние хранилища, и если там есть
   сохраненные данные, заполняй ими поля формы. В противном случае поля должны
   быть пустыми.
3. При сабмите формы очищай хранилище и поля формы, а также выводи объект с
   полями `email`, `password` и текущими их значениями в консоль.
4. Сделай так, чтобы хранилище обновлялось не чаще чем раз в 500 миллисекунд.
   Для этого добавь в проект и используй библиотеку
   [lodash.throttle](https://www.npmjs.com/package/lodash.throttle).

# Задание 3 - переключение темы

Напиши скрипт для изменения темы между светлой и темной при нажатии (событие
`change`) на чекбокс с идентификатором `theme-switch-toggle`.

- По умолчанию тема светлая.
- При изменении темы, необходимо добавлять на элемент `body` класс `light-theme`
  или `dark-theme` соответсвенно.
- Выбранная тема должна сохраняться при перезагрузке страницы. Для хранения темы
  используй локальное хранилище. Пусть ключ для хранилища называется `ui-theme`.
- Если при загрузке страницы тема тёмная, не забудь поставить свойство `checked`
  у чекбокса `#theme-switch-toggle` в `true`, чтобы ползунок сдвинулся в
  правильное положение.

Для удобства хранения списка классов используй такое перечисление.

```js
const theme = Object.freeze({
  light: 'light-theme',
  dark: 'dark-theme',
});
```
